requiremenets:

how to run:
  Diamond ...(put the path of ur file here) also put the path of where diamond is or it won't run unless you make it a global cmd

commands 7/22/26:
addp num1,num2 (NO SPACES after the ',')
subp num1,num2
divp num1,num2
mulp num1,num2
(p at the end means print)
also you can add {/} at the end without spaces to space a line
ex: addp [x],50{/} (to use variables values you put them inside [] )
also as for math cmds only addp can use variables rn

print .Hello world (after the . starts the string)
again you can do:
print .Hello world{/}
for variables
printv .X equals: [x]{/}

to declare variables 
only int types implemented yet
